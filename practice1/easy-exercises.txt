2. Sum 
    Write a program that asks the user for two numbers and then outputs(print's) the summed result to the console.
        Test Cases: 
        '3', '5' -> 8
        '2', '10' -> 12

3. Calculator
    Write a program that asks the user for two numbers and an operator (+, -, *, /) and performs the corresponding arithmetic operation on the numbers. Print the result to the console.
        Test Cases:
        '3', '5', '*' -> 15
        '10', '2', '/' -> 5

4. Fahrenheit to Celsius Converter
    Write a program that asks the user for a temperature in Fahrenheit and converts it to Celsius using the formula: celsius = (fahrenheit - 32) * 5/9. Print the result to the console.
    Test Cases:
        '33.8' -> 1 or (0.9999999999999984)
        '85' -> 29.4444

5. Check Odd or Even
Write a program that asks the user for an integer and checks whether it is odd or even. Print the result to the console.
